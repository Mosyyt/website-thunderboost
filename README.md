## What is ThunderBoost?

ThunderBoost is a service website for a Discord server boosting business — *"Giving power to the user."* It was originally hosted at `ethicalhax.xyz`. The site has a landing page, a bot info page, and a badges page. Purchases are handled through embedded Sellix product widgets. Stat counters use Odometer.js for animated number displays.

---

## Features

- **Landing page** — service description, pricing tiers and CTA
- **Sellix integration** — embedded product cards for direct purchase
- **Animated counters** — live stat numbers animated with Odometer.js
- **Bot page** (`bot.html`) — information about the associated Discord bot
- **Badges page** (`badges.html`) — server badge showcase
- **Bootstrap layout** — responsive grid system throughout
